# 🎓 Student Record Management System (C++)

A beginner-friendly console-based C++ project to manage student records using STL vectors.

## 🚀 Features

- Add new student  
- View all students  
- Menu-driven program  
- Uses C++ STL (vector)

## 🛠️ Tech Used

- C++  
- STL Vector  
- Structures  

## ▶️ How to Run

1. Compile the program  
2. Run the executable  
3. Use menu options to manage records  

## 📚 Learning Outcome

- Practiced structures in C++  
- Learned vector usage in real project  
- Improved menu-driven program logic
